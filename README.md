## 📌 About The Project

This project is a responsive FAQ Accordion built as part of a Frontend Mentor challenge.  
Users can click on a question to reveal or hide the answer dynamically.

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript (DOM Manipulation)

---

## ✨ Features

- Expand / Collapse FAQ answers
- Icon changes from plus ➕ to minus ➖
- Responsive layout
- Clean UI design
- Separated HTML, CSS, and JS files
